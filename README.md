# TheGreenNook
Kratki opis:

The Green Nook je projekt web stranice posvećene prodaji prirodnih proizvoda kao što su melemi, eterična ulja, kreme i čajevi. Cilj ovog projekta je dizajnirati modernu, preglednu i vizualno privlačnu web stranicu koja prezentira asortiman proizvoda i pruža osnovne informacije o firmi. Fokus je na kvalitetnom korisničkom sučelju i intuitivnoj navigaciji, bez implementacije funkcionalnosti online naručivanja. Stranica omogućava jednostavan pristup informacijama za krajnje korisnike, dok je dio za administraciju proizvoda predviđen kao koncept ili mockup. Ovaj projekt je prvenstveno usmjeren na aspekt dizajna web stranice, optimiziran za dobar prikaz na različitim uređajima.

Tehnologije korištene u radu:

Figma - za osnovni dizajn, dogovor oko izgleda stranice, te odabir slika i paleta boja koje će biti implementirane na samoj stranici.

HTML5 – za strukturu web stranice i organizaciju sadržaja.

CSS – za stilizaciju elemenata i oblikovanje korisničkog sučelja, uz korištenje fleksibilnih layout sistema poput Flexboxa i Grid-a.

JavaScript  – za osnovnu interaktivnost na stranici.

React.js – kao glavna biblioteka za izradu korisničkog sučelja, omogućava komponentni pristup i bolju organizaciju koda.

React Router – za implementaciju navigacije između različitih stranica unutar aplikacije bez potrebe za ponovnim učitavanjem.

Git i GitHub – za verzionisanje koda i timsku saradnju putem udaljenog repozitorija, i praćenje dokumantacije i napretka samog projekta.

WebStorm – razvojno okruženje (IDE) korišteno za pisanje i organizaciju koda.
